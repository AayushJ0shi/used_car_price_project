Project Overview

Smart Car Price Prediction System

AutoValue is an intelligent web application that accurately predicts used car prices using Machine Learning algorithms. Our system analyzes multiple parameters to provide real-time price estimations with 89.2% accuracy.

Developed as a B.Tech 7th Semester Minor Project, AutoValue demonstrates practical implementation of AI/ML concepts in real-world scenarios, bridging the gap between academic learning and industry requirements.

ML Pipeline & Methodology

1
Data Collection & Preprocessing

Gathered comprehensive car data including brand, age, mileage, engine specifications, and market prices. Cleaned and preprocessed data for ML readiness.

2
Feature Engineering

Created meaningful features like vehicle age from manufacturing year. Applied label encoding for categorical variables and normalized numerical features.

3
Model Training & Evaluation

Trained multiple algorithms including Random Forest and Linear Regression. Used cross-validation and metrics like R², RMSE, and MAE for evaluation.

4
Web Integration & Deployment

Integrated the best-performing model into Flask web application. Created REST API endpoints and deployed for real-world accessibility.
